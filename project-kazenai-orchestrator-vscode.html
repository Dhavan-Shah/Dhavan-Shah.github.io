Overview

Agent Builder is a multi-agent orchestrator for AI-assisted software development that emphasizes governance (approval gates) and cost awareness (budget + spend logging). The VS Code extension provides an IDE surface for running tasks, approving plans/code, and reviewing run telemetry.

The result is a workflow that mirrors real-world engineering constraints: structured planning, review stages, safe tool usage, and observable execution.
