Visualizing Outlier Detection using Deep Isolation Forest

  1. Home
  2. Projects
  3. Visualizing Outlier Detection

Project information

  • Category: Deep Learning, Outlier Detection
  • Project date: 20 March 2025

Overview

This project demonstrates an approach to outlier detection using a Deep Isolation Forest architecture with visualization tools to inspect decision paths and anomalous regions in high-dimensional data.

Frontend

The frontend includes interactive visualizations built with D3.js and React, allowing users to explore outlier detection results in real-time. The application provides interactive controls for dataset selection, batch processing, and threshold adjustment.

Technologies

  • Frontend: React, D3.js, Ant Design
  • Backend: Flask, Python
  • Algorithm: Deep Isolation Forest (DIF)
